Install in 3 Steps

No Chrome Web Store needed - load directly in Developer Mode

1

Download & Extract

Click "Download Extension" above and extract the ZIP to a folder of your choice (e.g. C:\qr-file-share-extension\).

2

Open Chrome Extensions

In Chrome, go to chrome://extensions and toggle Developer mode on (top-right switch). Then click "Load unpacked".

3

Select the Folder & Pin

Navigate to and select the extracted folder. The QR File Share extension appears in your extensions list. Click the Extension icon in Chrome → pin it to your toolbar. Done!
